Agentic AI & LLMs

DS-Crew: Multi-Agent Data Science Pipeline

GitHub
CrewAIScikit-learn / XGBoostOptuna MLflowPydantic

CrewAI-based multi-agent system that automates the full data-science lifecycle for structured/tabular data: EDA, cleaning, feature engineering, model selection, hyperparameter optimization, evaluation, and metadata logging. Agents propose actions while strictly-typed Python tools execute them, keeping raw data out of LLM context, with human-in-the-loop approval gates for cleaning plans, feature engineering, and final model selection. Layered validation (Pydantic schemas, dataset checks, task-level guardrails), a fixed model registry (XGBoost, LightGBM, CatBoost, scikit-learn) tuned via budget-capped Optuna search, and automatic MLflow experiment logging keep runs deterministic and reproducible. A headless AUTO_APPROVE mode supports unattended runs, with all auto-approvals tagged in MLflow.

Profile RAG Agent

GitHub Live Demo
FastAPIReact + ViteChroma FastEmbed (ONNX)Docker

Open-source RAG system for conversational profile discovery: candidates upload resumes, LinkedIn/Scholar text, GitHub links, and notes, and recruiters chat with the profile via a shareable URL with no installation required.
